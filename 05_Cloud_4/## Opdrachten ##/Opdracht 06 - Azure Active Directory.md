# Opdracht 6 - Azure Active Directory  

## Wat is Azure Active Directory

De Microsoft Azure Active Directory is eigenlijk een verlenging van je infrastructuur naar de cloud en werkt als een universeel identiteitsplatform om identiteiten te beheren en te beveiligen. Je behoudt bestaande on-premise investeringen en houdt voortaan controle vanuit de dynamische cloud.
